# DoctorWhoClassicRename
A little bash script to rename each episode of the Doctor Who classics to match with the format of https://thetvdb.com/

## Run

1. Put the script in the season folder, and execute : 

``./rename.sh SEASON_NUMBER(01, 02, 03 ..)``
